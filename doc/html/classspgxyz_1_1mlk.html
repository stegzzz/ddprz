Detailed Description

produce maximum likelihood ke and kh values.

for small or large rewards create with S1,S2 or L1,L2 respectively. computes indifference points along the way. these can be obtained with functions. Constructs base class lsk and uses least squares estimates as initial values. Uses ddgemModelNLopt
